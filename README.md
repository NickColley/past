## Quick start

Create a HTML file inside a 'pages' folder.

```bash
mkdir pages
echo '<h1>Hello, World</h1>' > pages/index.html
```

Then see your page by running `npx past`.

---

For all options run `npx past --help`.

See [./examples](./examples) for full list of examples.
