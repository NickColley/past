## Creating a page

Create a HTML file inside a 'pages' folder.

```bash
mkdir pages
echo '<h1>Hello, World</h1>' > pages/index.html
```

Then run `npx past`.

You will see your page at the URL [http://localhost:3000](http://localhost:3000).
